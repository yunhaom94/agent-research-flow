# Notes
*Notes are any other thoughts, observations, or additional information that is relevant to the project but does not related to a specific "idea". This can include things like background, random things to put in the paper, or anything else that is relevant to the project but does not fit into the "main ideas" or "minor ideas" categories. This should be a list of any sizes that can be changed as the project evolves.*
